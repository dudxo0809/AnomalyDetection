두 논문 모두 시계열 데이터를 사용했다.

-> 동일시간에 두개의 데이터가있는 데이터 (Multivariate time Series Data)

두 논문을 본 이유?

프로젝트가 비정상을 검출하기 위한 알고리즘 구현
	-> anomaly detection

두 논문 모두 오토 인코더를 사용해서 비정상 검출

오토인코더 : 입력데이터와 출력 데이터가 같은 모델

	-> 오토인코더로는 총 두가지 방식으로 검출가능 (논문당 하나)
두번째 논문의 방식
	
	-> 오토 인코더를 활용한 분류 (Convolutional auto encoder)
	
	-> 콘볼루션을 하는 이유 : feature을 검출하기 위함
	-> 보틀넥에는 농축된 feature가 존재 (최적화된)

	-> 첫번째 방식은 농축된 feautre을 보고 encoder단계의 데이터가 어떤 특성을 지니는지 분류하는 모델


첫번째 논문의 방식

	-> 정상 데이터만 LSTM 오토인코더 모델에 넣어 훈련시키면
	-> 후에 비정상 데이터가 입력시 데이터의 차이가 크게됨
		-> 비정상 검출


결론은 

	-> 시계열 데이터에서 비정상을 검출하는 알고리즘 두가지를 확인했다.
